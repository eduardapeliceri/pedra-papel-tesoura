# pedra-papel-tesoura
Jogo simples de Pedra, Papel e Tesoura contra a Harley Quinn, desenvolvido com HTML, CSS e JavaScript.

## 🛠 Tecnologias usadas
- HTML
- CSS
- JavaScript

## 💡 Funcionalidades
- Escolha entre Pedra, Papel ou Tesoura
- Jogue contra a "máquina" (Harley Quinn)
- Placar dinâmico de vitórias
